Projects

Here, there is a summary of my main coding projects. All the following projects and even more are all available on my codeberg page (an open-source GitHub alternative): timmnx.

Symbolic Execution guided by A* - OCaml

Project carried out as part of my L3 internship with Benjamin FARINIER from team EPICURE at IRISA (Rennes, France).

The Abstract Interpreter coded by Benjamin Farinier raises an invariant potentially responsible for a bug in the examined code. My Symbolic Executor uses this invariant, transformed into a logical formula, as the target for an A* on states.

PROLOG Inferator - OCaml

Project carried out as part of a 3-day practical course at ENS Rennes, in groups of three.

The code infers PROLOG rule sets and then determines the truth of a predicate or assigns the free variables left to make it true.

Fractals of Julia and Mandelbrot - C

Project carried out in parallel of my MP2I then completed in MPI*.

Displays Julia and Mandelbrot sets. The rendering is generated using the SDL library and computations are accelerated with the pthread library.

Artificial Intelligence for Othello Game - Python

Project carried out during my 12th grade / Year 13 for my participation to "Les Trophées NSI" (Edition 2023).

Othello board game where you play against the computer that uses a Min-Max algorithm.
